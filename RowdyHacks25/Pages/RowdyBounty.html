This page demonstrates using the Rio Grande display font via @font-face. Replace the font files in ./fonts/ with the TTF/WOFF/WOFF2 files you downloaded.

Heading set with Rio Grande

Rio Grande is a bold, decorative display font best for headlines and logos — not for long body text. Use a readable serif or sans-serif for paragraphs. The CSS above uses font-display: swap to avoid invisible text while the font loads.

Example body copy (system font fallback) — keep long paragraphs in system or web-safe fonts for readability.
